<h1 align="center">Cognisense – Palmistry Analyzer</h1>

<p align="center">
An AI-powered palm analysis system that interprets personality traits and behavioral patterns using computer vision.
</p>

<hr>

<h2>Project Overview</h2>
<p>
Cognisense Palmistry Analyzer is a computer vision based application that analyzes palm images and extracts meaningful
patterns from major palm lines. The system detects hand landmarks, identifies palm features, and generates structured
personality interpretations using rule-based logic.
</p>

<p>
The project demonstrates the integration of Artificial Intelligence with traditional palm analysis concepts through
modern machine learning tools.
</p>

<hr>

<h2>Features</h2>
<ul>
<li>Palm detection and landmark extraction</li>
<li>Automatic palm line identification</li>
<li>Rule-based personality interpretation</li>
<li>Interactive web interface</li>
<li>Image upload and processing</li>
<li>Visual output with predictions</li>
</ul>

<h2>Installation</h2>

<h3>Clone Repository</h3>
<pre>git clone https://github.com/GazanfarAnsari10/Cognisense-Palmistry-Analyzer.git</pre>

<h3>Create Virtual Environment</h3>
<pre>python -m venv venv</pre>

<h3>Activate Environment (Windows)</h3>
<pre>venv\Scripts\activate</pre>

<h3>Install Dependencies</h3>
<pre>pip install -r requirements.txt</pre>

<h3>Run Application</h3>
<pre>python app.py</pre>

<hr>

<h2>How It Works</h2>
<ul>
<li>User uploads a palm image</li>
<li>MediaPipe detects hand landmarks</li>
<li>Lines and patterns are extracted</li>
<li>Rule-based logic interprets traits</li>
<li>Results displayed on web interface</li>
</ul>
